# Whiplash

Whiplash is a multichannel order fulfillment and third-party logistics (3PL) platform providing a REST API for managing orders, SKUs, inventory levels, shipments, and returns across warehouse locations. Whiplash was acquired by Ryder System for approximately $480 million and now operates as RyderShip, Ryder's ecommerce fulfillment platform.

The Whiplash/RyderShip API enables developers to automate fulfillment operations programmatically — creating orders, managing inventory, tracking shipments, processing returns, and receiving event-driven notifications via webhooks.
